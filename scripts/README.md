# สร้างแอดมินคนแรก

Script นี้ใช้สร้างบัญชีแอดมินคนแรกสำหรับระบบ

## ขั้นตอนการใช้งาน

### 1. ติดตั้ง dependencies
```bash
npm install firebase
```

### 2. รัน script
```bash
node scripts/create-admin.js
```

### 3. กรอกข้อมูล
- อีเมลแอดมิน
- รหัสผ่าน (อย่างน้อย 6 ตัวอักษร)
- ชื่อผู้ใช้

### 4. ลบ script หลังใช้งาน
```bash
rm scripts/create-admin.js
```

## ความปลอดภัย
- Script นี้ควรใช้ครั้งเดียวเพื่อสร้างแอดมินคนแรก
- หลังใช้งานควรลบทิ้งเพื่อความปลอดภัย
- แอดมินที่สร้างจะมีสิทธิ์ครบถ้วนทุกฟังก์ชัน

## สิทธิ์ของแอดมิน
- user_management (จัดการผู้ใช้)
- inventory_management (จัดการสต็อก)
- order_management (จัดการคำสั่งซื้อ)
- system_settings (ตั้งค่าระบบ)
